API Keys

Create API keys for programmatic access to the todo.mdx API. Use these in CI/CD pipelines or other automation.

Please sign in to manage your API keys.

Usage

# Environment variable
export TODOMDX_API_KEY=sk_live_xxx

# SDK usage
import { createClient } from 'todo.mdx'

const client = createClient({
  repo: 'owner/repo',
  apiKey: process.env.TODOMDX_API_KEY,
})

const issues = await client.issues.list()
