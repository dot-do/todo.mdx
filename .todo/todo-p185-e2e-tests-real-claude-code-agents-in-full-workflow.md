# E2E Tests: Real Claude Code Agents in Full Workflow

End-to-end tests that run actual Claude Code instances through the complete autonomous development cycle:

1. Issue becomes ready (blocker closes)
2. Claude develops the implementation
3. Claude creates PR
4. Claude requests review
5. Reviewer requests changes
6. Claude addresses feedback
7. Claude re-requests review
8. Reviewer approves
9. Claude merges PR
10. Issue closes
