# README.md and CLAUDE.md show incorrect file naming pattern

The README.md (line 48) states that issue files use the pattern `.todo/{id}-{slug}.md`, but the actual implementation in `src/generator.ts` uses `[yyyy-mm-dd] [Title].md` as the DEFAULT_PATTERN.

**Locations:**
- `/Users/nathanclevenger/projects/todo.mdx/README.md` line 48
- `/Users/nathanclevenger/projects/todo.mdx/docs/configuration.md` line 21 (mentions `{id}-{title}.md`)

**Actual behavior:**
Files are named like: `2025-12-24 Fix Cli Version Mismatch 0.1.0 Vs 0.2.0.md`

This would confuse users trying to understand the file naming convention.
