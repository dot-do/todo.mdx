# Add xterm.js assets to worker/public

Add xterm.js and xterm-addon-fit to the worker's public assets for the terminal widget pages.

## Files to create:
- `worker/public/assets/xterm.min.js` - xterm.js library
- `worker/public/assets/xterm.min.css` - xterm.js styles
- `worker/public/assets/xterm-addon-fit.min.js` - fit addon
- `worker/public/assets/terminal.js` - our terminal wrapper code

## terminal.js should:
- Initialize xterm.js with proper theme
- Connect to WebSocket
- Handle resize with fit addon
- Send/receive data
- Handle disconnect/reconnect
