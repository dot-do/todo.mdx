# Create .env.example for E2E test configuration

Document all required environment variables for running E2E tests.

## File: tests/.env.example
```
# Worker API
WORKER_BASE_URL=https://todo.mdx.do
WORKER_ACCESS_TOKEN=your_access_token

# MCP Server
MCP_BASE_URL=https://todo.mdx.do
MCP_ACCESS_TOKEN=your_mcp_token

# GitHub App (for GitHub integration tests)
GITHUB_APP_ID=your_app_id
GITHUB_PRIVATE_KEY=base64_encoded_private_key
GITHUB_INSTALLATION_ID=your_installation_id
GITHUB_WEBHOOK_SECRET=your_webhook_secret

# Linear (optional, for Linear integration tests)
LINEAR_API_KEY=your_linear_api_key
LINEAR_TEAM_ID=your_team_id

# Sites (optional, defaults shown)
SITE_BASE_URL=https://todo.mdx.do
ADMIN_BASE_URL=https://admin.mdx.do

# Test fixtures
FIXTURES_DIR=./fixtures/test.mdx
```
