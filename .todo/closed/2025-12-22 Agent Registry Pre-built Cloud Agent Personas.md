# Agent Registry: Pre-built Cloud Agent Personas

Pre-built named agents available in the cloud, customizable via agents.mdx.

## Agent Roster
| Agent | Focus |
|-------|-------|
| **Product Priya** | Roadmap, planning, prioritization, assignment |
| **Coder Cody** | General development |
| **Typescript Tom** | TS/JS specialist |
| **Quinn QA** | Testing, quality |
| **Security Sam** | Security review, audits |
| **Docs Dana** | Documentation |

## Features
- Pre-built agents work out of the box
- agents.mdx can customize capabilities, focus areas, autonomy
- Cloud stores runtime state (sessions, capacity)
- Dashboard shows agent status
- Org/repo can define custom agents
