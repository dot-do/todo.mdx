# PRDO: PR Code Review SDLC

Implement a Durable Object (PRDO) that manages the complete code review lifecycle for pull requests using XState. When an agent opens a PR, configured reviewers (Priya, Quinn, Sam) review sequentially. If changes are requested, the original author agent is dispatched to fix issues, and the review cycle repeats until all approve.
