# .github/README.md webhook URLs mismatch app-manifest.json

The webhook and redirect URLs documented in .github/README.md do not match what is configured in app-manifest.json:

**app-manifest.json (actual config):**
- Webhook URL: `https://todo.mdx.do/api/webhooks/github`
- Redirect URL: `https://todo.mdx.do/github/callback`

**.github/README.md lines 38-39 (documented):**
- Webhook URL: `https://api.todo.mdx.do/github/webhook`
- Redirect URL: `https://api.todo.mdx.do/github/callback`

The documented URLs use `api.todo.mdx.do` subdomain while the actual manifest uses `todo.mdx.do`. Also the webhook path differs (`/github/webhook` vs `/api/webhooks/github`).

This inconsistency would cause confusion when setting up the GitHub App.
