# Repository Agents

This file defines AI agents for this repository. Agents are synced to the cloud on push and can be assigned to issues for autonomous development.

## Code Reviewer

<Agent
  name="Code Reviewer"
  description="Reviews pull requests for code quality, security, and best practices"
  autonomy="supervised"
  model="sonnet"
  focus={['code-quality', 'security', 'testing']}
  capabilities={[
    { name: 'git', operations: ['read', 'diff'] },
    { name: 'github', operations: ['comment', 'review', 'request-changes'] },
    { name: 'claude', operations: ['ask', 'research'] }
  ]}
  triggers={[
    { event: 'pull_request.opened', condition: 'draft === false' },
    { event: 'pull_request.synchronize' }
  ]}
/>

## Test Builder

<Agent
  name="Test Builder"
  description="Writes comprehensive tests for new features"
  autonomy="full"
  model="sonnet"
  focus={['testing', 'tdd', 'quality-assurance']}
  capabilities={[
    { name: 'git', operations: ['*'] },
    { name: 'npm', operations: ['test', 'build'] },
    { name: 'claude', operations: ['do', 'research'] }
  ]}
  triggers={[
    { event: 'issue.ready', condition: "labels.includes('needs-tests')" }
  ]}
/>

## Usage

When you push this file to your repository, these agents will be synced to the cloud. You can then:

1. **Assign agents to issues**: Add the agent name to the issue's assignee field
2. **View agent status**: Check the dashboard for active sessions and capacity
3. **Override global agents**: Agents defined here override built-in agents with the same name
4. **Customize per repo**: Each repo can have its own agent configurations
