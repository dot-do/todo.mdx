## Prerequisites

- Node.js 20 or later
- npm, pnpm, or yarn

## Install Package

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tab value="npm">
    ```bash
    npm install claude.mdx
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add claude.mdx
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add claude.mdx
    ```
  </Tab>
</Tabs>

## Initialize

Create a `TODO.mdx` file in your project root:

```mdx
# TODO

## Ready to Work
<Issues.Ready limit={5} />
```

## Configure

Add to your `package.json`:

```json
{
  "scripts": {
    "todo": "claude.mdx compile TODO.mdx"
  }
}
```

## Verify Installation

Run the compiler to generate `TODO.md`:

```bash
npm run todo
```

You should see a `TODO.md` file with your rendered issues.
