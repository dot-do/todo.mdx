## Configuration File

Create a `claude.mdx.config.ts` file in your project root:

```typescript
import { defineConfig } from 'claude.mdx'

export default defineConfig({
  sources: {
    beads: {
      enabled: true,
      path: '.beads',
    },
    github: {
      enabled: true,
      repo: 'owner/repo',
      token: process.env.GITHUB_TOKEN,
    },
    payload: {
      enabled: true,
      url: process.env.PAYLOAD_URL,
      token: process.env.PAYLOAD_TOKEN,
    },
  },
  output: {
    directory: '.',
    files: ['TODO.md', 'ROADMAP.md'],
  },
})
```

## Data Sources

### beads (Local)

Use local beads issue tracker:

```typescript
{
  beads: {
    enabled: true,
    path: '.beads', // Path to beads directory
  }
}
```

### GitHub Issues

Sync with GitHub Issues:

```typescript
{
  github: {
    enabled: true,
    repo: 'owner/repo',
    token: process.env.GITHUB_TOKEN,
  }
}
```

### Payload CMS

Use Payload CMS as data source:

```typescript
{
  payload: {
    enabled: true,
    url: process.env.PAYLOAD_URL,
    token: process.env.PAYLOAD_TOKEN,
  }
}
```

## Output Options

Control where and how files are generated:

```typescript
{
  output: {
    directory: '.',
    files: ['TODO.md', 'ROADMAP.md', 'CLAUDE.md'],
    watch: true, // Auto-regenerate on changes
  }
}
```

## Environment Variables

Create a `.env` file:

```bash
GITHUB_TOKEN=ghp_your_token_here
PAYLOAD_URL=https://your-payload.com
PAYLOAD_TOKEN=your_payload_token
```

## Watch Mode

Auto-regenerate markdown when MDX files change:

```bash
claude.mdx compile --watch TODO.mdx
```

## CI/CD Integration

Add to your CI pipeline:

```yaml
- name: Generate documentation
  run: |
    npm install
    npm run todo
    git add TODO.md ROADMAP.md
    git commit -m "docs: update generated documentation"
    git push
```
