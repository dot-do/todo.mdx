# Terminal Rendering

cli.mdx uses React and Ink to render MDX components in the terminal with rich formatting.

## How It Works

1. **Parse MDX** - Your `.mdx` file is compiled to JavaScript
2. **Render React** - Components render using React (via Ink)
3. **Terminal Output** - Ink converts React to ANSI terminal codes

## Styling

Use standard MDX for rich terminal output:

```mdx
# Heading

**Bold text** and *italic text*

- Bullet points
- Look great
- In the terminal

> Blockquotes are highlighted

\`\`\`bash
code blocks
are syntax highlighted
\`\`\`
```

## Colors

Import color utilities:

```mdx
import { Text } from 'cli.mdx'

<Text color="green">Success!</Text>
<Text color="red" bold>Error!</Text>
<Text color="yellow" italic>Warning</Text>
```

## Layout

Use Ink components for layout:

```mdx
import { Box, Text } from 'cli.mdx'

<Box flexDirection="column" padding={1}>
  <Text bold>Title</Text>
  <Text>Description text</Text>
</Box>

<Box flexDirection="row" gap={2}>
  <Text>Left</Text>
  <Text>Right</Text>
</Box>
```

## Interactive Elements

Create interactive prompts:

```mdx
import { Select, TextInput } from 'cli.mdx'

<Select
  label="Choose an option:"
  options={[
    { label: 'Option 1', value: '1' },
    { label: 'Option 2', value: '2' }
  ]}
/>

<TextInput label="Enter your name:" />
```

## Progress

Show progress indicators:

```mdx
import { Spinner, ProgressBar } from 'cli.mdx'

<Spinner label="Loading..." />

<ProgressBar total={100} current={progress} />
```
