# Examples

Learn from real-world examples of agents built with agents.mdx.

## Code Review Agent

An agent that reviews pull requests:

```mdx
---
name: code-reviewer
description: Automated code review agent
---

# Code Review Agent

<Agent.Context>
  <Source type="codebase" path="./src" />
  <Source type="docs" url="https://standards.company.com" />
</Agent.Context>

<Agent.Workflow>
  <Step name="fetch-pr">Fetch pull request changes</Step>
  <Step name="analyze">Analyze code quality</Step>
  <Step name="check-tests">Verify test coverage</Step>
  <Step name="comment">Post review comments</Step>
</Agent.Workflow>
```


## CI/CD Agent

Manage continuous integration and deployment:

```mdx
---
name: ci-cd
description: Automated CI/CD pipeline
---

# CI/CD Agent

<Agent.Context>
  <Source type="codebase" path="." />
  <Source type="api" url="https://api.deployment.com" />
</Agent.Context>

<Agent.Workflow>
  <Step name="build">Build the project</Step>
  <Step name="test" parallel>Run test suite</Step>
  <Step name="lint" parallel>Run linters</Step>
  <Step name="security" parallel>Security scan</Step>
  <Step name="deploy" depends={['build', 'test', 'lint', 'security']}>
    Deploy to staging
  </Step>
  <Step name="verify">Verify deployment</Step>
  <Step name="promote" condition={verified}>
    Promote to production
  </Step>
</Agent.Workflow>
```

## Release Notes Agent

Generate release notes from commits:

```mdx
---
name: release-notes
description: Generate release notes
---

# Release Notes Agent

<Agent.Context>
  <Source type="git" path="." />
  <Source type="issues" repo="owner/repo" />
</Agent.Context>

<Agent.Workflow>
  <Step name="collect">Collect commits since last release</Step>
  <Step name="categorize">Categorize changes</Step>
  <Step name="link">Link to issues and PRs</Step>
  <Step name="format">Format release notes</Step>
  <Step name="publish">Publish release notes</Step>
</Agent.Workflow>
```
