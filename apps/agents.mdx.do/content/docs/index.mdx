# agents.mdx

**AI Agent Framework in MDX**

Define AI agent behavior, context, and workflows in markdown. Build intelligent agents that understand your codebase and domain.

## What is agents.mdx?

agents.mdx is a framework for building AI agents using MDX (Markdown + JSX). It allows you to:

- Define agent behavior in readable markdown
- Specify context and knowledge sources
- Create workflows and task sequences
- Build interactive agent interfaces

## Quick Start

Install the package:

```bash
npm install agents.mdx
# or
pnpm add agents.mdx
```

Create your first agent:

```mdx
---
name: code-reviewer
description: Review code for best practices
---

# Code Review Agent

This agent reviews code and provides feedback on:

- Code quality and style
- Performance considerations
- Security best practices
- Documentation completeness

## Context

<Agent.Context>
  <Source type="codebase" path="./src" />
  <Source type="docs" url="https://docs.example.com" />
</Agent.Context>

## Workflow

<Agent.Workflow>
  <Step name="analyze">Analyze the code changes</Step>
  <Step name="check-tests">Verify test coverage</Step>
  <Step name="suggest">Provide improvement suggestions</Step>
</Agent.Workflow>
```

## Features

### MDX-First Design

Write agents in markdown with embedded React components for interactive elements.

### Context Management

Define and manage agent context from multiple sources:
- Local codebases
- Documentation sites
- API endpoints
- Database queries

### Workflow Automation

Create multi-step workflows that agents follow automatically.

### Type-Safe

Full TypeScript support for agent definitions and configurations.
