# CLI.mdx

MDX-based CLI framework

**28 open** · 6 in progress · 23 closed · 57 total (40% complete)


## Commands

**render**
  Render MDX to terminal

**build**
  Compile MDX to markdown file

**dual**
  Render to both terminal and markdown


## Features

- Define CLI commands as MDX components
- React components that render to terminal
- Same components render to markdown files
- Live data injection from beads, GitHub, APIs
- Multi-output: terminal, README.md, CLAUDE.md
