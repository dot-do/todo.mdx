## Project Context

todo.mdx is an MDX-based project management system. Key concepts:

- **MDX templates** - Define views (TODO.mdx, ROADMAP.mdx, AGENTS.mdx)
- **Compilation** - MDX → markdown with live data injection
- **Data sources** - beads (.beads/), GitHub Issues, package.json, MCP tools
- **Multi-output** - Single source → multiple formats (CLAUDE.md, .cursorrules, etc.)

## Architecture

```
packages/
├── todo.mdx/      - TODO.mdx → TODO.md compiler
```
